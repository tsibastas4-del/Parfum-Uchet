# Інструкція: Підключення Supabase

1.  **Реєстрація/Вхід**
    - Зайдіть на [supabase.com](https://supabase.com) і натисніть "Start your project".
    - Увійдіть через GitHub (або зареєструйтесь).

2.  **Створення Проекту**
    - Натисніть **"New Project"**.
    - Name: `PerfumeCRM` (або як хочете).
    - Password: придумайте складний пароль (він нам зараз не знадобиться, але збережіть).
    - Region: `Frankfurt` (найближчий) або будь-який.
    - Натисніть **"Create new project"**. Зачекайте пару хвилин.

3.  **Отримання Ключів**
    - Коли проект створиться, ви побачите сторінку налаштувань.
    - Знайдіть розділ **API Settings** (іконка шестерні -> API).
    - Вам треба два значення:
        - `Project URL` (виглядає як `https://xyz...supabase.co`)
        - `Project API Key` (виглядає як довгий набір букв `eyJ...`, позначений як `anon` `public`).
    - **Не копіюйте `service_role` ключ!** Тільки `anon`.

4.  **Налаштування Таблиці**
    - Зліва в меню оберіть **Table Editor** (іконка таблиці).
    - Натисніть **"New Table"**.
    - Name: `app_data`
    - Вимкніть галочку **"Enable Row Level Security (RLS)"** (Для простоти, щоб працювало одразу. Пізніше можна налаштувати).
    - В колонках:
        - `id` (залиште int8, Primary Key) - воно дефолтне.
        - Натисніть **"Add Column"**:
            - Name: `json_data`
            - Type: `jsonb`
    - Натисніть **"Save"**.

5.  **Підключення в Програмі**
    - Відкрийте нашу CRM.
    - Натисніть ⚙️ (Налаштування) вгорі праворуч.
    - Вставте **URL** та **Key**.
    - Натисніть **"Перевірити з'єднання"**.
    - Якщо зелене — натисніть **"Зберегти в хмару"**.
